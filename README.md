# ad9833_functiongenerator
this is a portable function generator code for the ad9833 complete with all libraries. complete with display for seeing the current frequency and rotary encoders for chaniging the frequency output, 3 waveform modes-- sine , triangle, square. used freeRTOS for task distribution but primarily for learning the platform for concurrent programing

#  my test bench
built the entire thing on breadbord , migrating to a better suited mcu on a pcb soon
programmer - stlinkv2
middlewhere used - freeRTOS
hardware used- 
1. LCD screen 16x2(parallel)mode
2. ad9833 dds signal generator
3. rotary encoders for frequency changing
4. buttons for chaing modes and frequency increment per turn of the rotary
